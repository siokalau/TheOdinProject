On this page are all my answers to the questions asked in the Knowledge Check secttion

1. Introduction to HTML

  • What do HTML and CSS stand for?

    • Hypertext Markup Language and Cascading Style Sheet

  • Between HTML and CSS, which would you use for putting paragraphs of text on a webpage?

    • I would use HTML, because it is responsible for the structure of a website.

  • Between HTML and CSS, which would you use for changing the font and background color of a button?

    • I would use CSS, because it is used for customizing the looks and design.

  • What is the difference between HTML, CSS and JavaScrit?

    • HTML is used for the structure of the webiste, CSS for the looks and JavaScript is used to add interactivity to the other two.

2. Elements and Tags

  • What is an HTML tag?

    • HTML tags are a part of the elements, they define the structure and content of a webpage.

  • What are the three parts of an HTML element?
    1. The opening tag
    2. The content
    3. The closing tag

  • What are void elements, and how are they different from regular HTML elements?

    • Void elements don't have a closing tag, meaning that they do not wrap content, so they have nothing inside them.

3. HTML Boilerplate

  • What is the purpose of the doctype declaration?

    • The doctype declaration is used to tell the browser what version of HTML should be used for rendering the document.

  • What is the HTML element?

    • It is the most important element of the document, all the other elements will be descendants of it.

  • What is the purpose of the head element?

    • The head element is not displayed on the page, we shoud only have information that tells the browser how to render the pages correctly.

  • What is the purpose of the body element?

    • The body element is used to display all of the stuff that we want the users to see.
