On this page are all my answers to the questions asked in the Knowledge Check secttion

1. Introduction to HTML

  • What do HTML and CSS stand for?

    • Hypertext Markup Language and Cascading Style Sheet

  • Between HTML and CSS, which would you use for putting paragraphs of text on a webpage?

    • I would use HTML, because it is responsible for the structure of a website.

  • Between HTML and CSS, which would you use for changing the font and background color of a button?

    • I would use CSS, because it is used for customizing the looks and design.

  • What is the difference between HTML, CSS and JavaScrit?

    • HTML is used for the structure of the webiste, CSS for the looks and JavaScript is used to add interactivity to the other two.

2. Elements and Tags

  • What is an HTML tag?

    • HTML tags are a part of the elements, they define the structure and content of a webpage.

  • What are the three parts of an HTML element?
    1. The opening tag
    2. The content
    3. The closing tag

  • What are void elements, and how are they different from regular HTML elements?

    • Void elements don't have a closing tag, meaning that they do not wrap content, so they have nothing inside them.

3. HTML Boilerplate

  • What is the purpose of the doctype declaration?

    • The doctype declaration is used to tell the browser what version of HTML should be used for rendering the document.

  • What is the HTML element?

    • It is the most important element of the document, all the other elements will be descendants of it.

  • What is the purpose of the head element?

    • The head element is not displayed on the page, we shoud only have information that tells the browser how to render the pages correctly.

  • What is the purpose of the body element?

    • The body element is used to display all of the stuff that we want the users to see.

4. Working with text

  • How do you create a paragraph in HTML?

    • To create a paragaph you use the <p> opening tag and the </p> closing tag, with the desired contetnt written between them

  • How do you create a heading in HTML and how many levels of headings ar there?

    • A heading is <h1> to <h6>.

  • What is the difference between the levels of headings.

    • The difference is the size of the font and also they are ranked by importance from h1 being the most important, to h6 being the least.

  • What should you use to make text bold and important?

    • In this case I should use <strong>

  • What element should you use to make text italicized to add emphasis to it?

    • I should use <em>

  • What relationship does an element have with any nested elements within it?

    • It has a parent-child connection.

  • What relationship do two elements have if they are at the same level of nesting?

    • They have a sibiling relationship.

5. Lists

  • What HTML element is used to create an unordered list?

    • The <ul> element is used

  • What HTML element is used to create an ordered list?

    • The <ol> element is used.

  • What HTML element is used to create list items within both unordered and ordered lists?

    • The <li> item is used in this case.

6. Links and Images

  • What element is used to create a link?

    • We use the <a> element to create a link.

  • What is an attribute>

    • An attribute give more information about an element.

  • What attribute tells links where to go to?

    • The href= attribute.

  • What security considerations must be taken if you wish to use the target attribute to open links in a new tab/window?

    • You should use the rel="noopener noreferrer" to avoid phishing attacks and if you dont want the opened link to know that our webpage links it.

  • What is the difference between an absolute and relative link?

    • An absolute one links to an external website, while the relative one links to a page that is stored in the same website's folders.

  • Which element is used to display an image?

    • The <img> element, using a src= attribute and an alt= attribute

  • How do you access a parent directory in a filepath?

    • By using ../

  • What are the four main image formats that you can use for images on the web?

    • You can use JPG, PNG, GIF or SVG
